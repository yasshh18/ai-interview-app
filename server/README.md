# 🎯 AI Interviewer

An AI-powered mock interview platform that simulates real interview conversations using voice interaction.

## 🚀 Features

- 🎤 Real voice conversation with AI interviewer
- 🤖 Powered by Groq (LLaMA 3.3) for intelligent responses
- 🔊 Human-like voice using ElevenLabs TTS
- 👤 Secure authentication with JWT & bcrypt
- 💼 Multiple roles: Software Engineer, DevOps, Cloud Engineer
- 🌐 English & Hindi language support
- 📱 Beautiful dark UI

## 🛠️ Tech Stack

**Frontend:** React.js, Tailwind CSS, Web Speech API

**Backend:** Node.js, Express.js, MongoDB Atlas

**AI:** Groq API (LLaMA 3.3 70B), ElevenLabs TTS

**Auth:** JWT, bcrypt

## ⚙️ Setup

1. Clone the repo
2. Install dependencies
3. Add environment variables
4. Run the app
